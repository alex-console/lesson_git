<-----НАЧАЛО----->

Git init - это инициализация проекта и гит теперь следит за данным репозиторием
Git remote add origin (url git) - связываем локальный репозиторий с удаленным
Git clone (url git) - копирование с удаленного репозитория к себе в локальный репозиторий проект

<-----КОММИТЫ----->

Git status - проверка на коммит
Git add . - подготовка всех файлов к коммиту
Git commit -m "massage commit" - коммит с комментарием
